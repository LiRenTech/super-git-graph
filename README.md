这是一个基于tauri+React的桌面应用项目，

用于在一个无限大的二维平面中查看项目的git graph。

操作方式：

使用鼠标滚轮缩放
中键拖拽移动视野。
鼠标左键框选节点，左键拖拽移动节点。
右键可以弹出右键菜单。

用户可以自定义更改git节点图的布局，布局会被持久化的保存，再次打开时，会加载上次保存的布局。

不同的分支会以不同的颜色显示。

stash的节点会以双圈显示。

分支会以一个指向节点的标签的样子显示在图上。

这个应用有标签页功能，能同时打开多个项目的git graph。

git grpah页面具有刷新按钮，可以一键刷新当前项目的git graph。

git graph页面具有一个搜索框，用户可以在搜索框中输入关键词，搜索框会实时显示搜索结果。

git graph页面中的节点群具有懒加载功能，目前只显示较近的节点，早起的节点会省略，用户可以通过点击点点点来加载早期的节点
